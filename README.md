# Employee-Tracker

## Author: Sudhanshu Batra

## Description
Employee Tracker allows an Admin user t View Departments Employees and Roles that are set within the company. The application also allows the user to view salaries of each role and view Job Designations of employees across the board. The application enables the user to Add new Department. Add new Employees, Add new Roles and assign Manages to new Employees. The application also allows the user to update and existing employee profile. When finished with the application, user can Quir the application by selecting the Quit option.
